Ollama

Local Large Language Model Runner

üìã System Requirements

  • RAM: 8GB minimum, 16GB+ recommended
  • Storage: 4GB+ free space per model
  • GPU: Optional but recommended (NVIDIA/AMD)

1Install Ollama

# Windows Installation # Download installer from: https://ollama.ai/download # Run the .exe installer and follow prompts # macOS Installation # Method 1: Download from https://ollama.ai/download # Method 2: Using Homebrew brew install ollama # Linux Installation curl -fsSL https://ollama.ai/install.sh | sh

2Download and Run Models

# Start Ollama service (runs automatically on Windows/Mac) ollama serve # Download popular models ollama pull llama3.2:3b # 2GB - Fast, good for coding ollama pull llama3.1:8b # 4.7GB - Balanced performance ollama pull codellama:7b # 3.8GB - Code-focused model # Run a model interactively ollama run llama3.2:3b # List installed models ollama list

‚úÖ Verify Installation

# Check Ollama version ollama --version # Test with a simple prompt ollama run llama3.2:3b "Hello, how are you?" # API test (in another terminal) curl http://localhost:11434/api/generate -d '{ "model": "llama3.2:3b", "prompt": "Why is the sky blue?" }'

Node.js

JavaScript Runtime for Server-Side Development

üìã System Requirements

  • OS: Windows 10+, macOS 10.15+, Linux (most distributions)
  • RAM: 512MB minimum, 2GB+ recommended
  • Storage: 200MB for Node.js installation

1Install Node.js

‚ö†Ô∏è Recommendation: Install the LTS (Long Term Support) version for stability
# Method 1: Official Installer # Visit: https://nodejs.org # Download LTS version (recommended) # Run installer and follow prompts # Method 2: Package Manager # Windows (using Chocolatey) choco install nodejs # Windows (using winget) winget install OpenJS.NodeJS # macOS (using Homebrew) brew install node # Ubuntu/Debian sudo apt update sudo apt install nodejs npm # CentOS/RHEL/Fedora sudo dnf install nodejs npm

2Alternative: Node Version Manager (Recommended)

# Install NVM (Node Version Manager) # Windows - Install nvm-windows # Download from: https://github.com/coreybutler/nvm-windows # macOS/Linux - Install nvm curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash # Restart terminal, then: nvm install --lts # Install latest LTS nvm use --lts # Use latest LTS nvm install 18.17.0 # Install specific version nvm use 18.17.0 # Switch to specific version
